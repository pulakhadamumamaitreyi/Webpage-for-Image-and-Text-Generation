# 🌐 Pear Media Assignment – Text & Image Generation Web App

## 📌 Project Overview

This project is a **web-based prototype** that integrates multiple APIs to perform:

* ✨ Text enhancement using NLP
* 🎨 Image generation from text prompts
* 🖼 Image analysis from uploaded images

The application demonstrates **API integration, prompt engineering, and full-stack development** within a limited timeframe.

## 🔄 Application Workflow

### ✏️ Text Input Workflow

1. User enters a text prompt
2. System enhances the prompt using NLP
3. Enhanced prompt is displayed
4. User generates image from enhanced prompt
5. Image is displayed

---

### 🖼 Image Input Workflow

1. User uploads an image
2. System analyzes the image (objects, theme, description)
3. Description is displayed to the user

## ⚙️ Setup Instructions

### 🔹 Prerequisites

* Node.js installed
* OpenAI API Key

---

### 🔹 Backend Setup

```bash
cd backend
npm install
```

Create a `.env` file:

```
OPENAI_API_KEY=your_api_key_here
```

Run server:

```bash
node server.js
```

Server runs at:

```
http://localhost:3000
```

---

### 🔹 Frontend Setup

Open the file:

```
frontend/index.html
```

in your browser.
